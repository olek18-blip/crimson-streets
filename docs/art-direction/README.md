# Art direction

This folder defines the visual target for **Crimson Streets**.

## Purpose

These documents turn visual references into production rules that can be implemented in the game.

They are intended to guide:

- character silhouettes
- faction readability
- color language
- environment mood
- UI/HUD cohesion
- future Godot asset production

## Files

- `visual-target.md` — overall visual identity and rendering target
- `character-style-guide.md` — playable character, gangs, police, civilians, and support archetypes
