# Godot v2 Status

This is the current recommended Godot 4 base for The Dangerous Spain / Crimson Streets.

Open this project in Godot 4.6:

```text
godot_v2/project.godot
```

Main scene:

```text
res://scenes/Main.tscn
```

Current playable slice:

- data-driven Barcelon blockout from `data/barcelon_layout.json`
- player movement with gravity
- follow camera
- enter and exit a simple drivable car with `F`
- pooled lane traffic from the layout JSON
- first mission marker loop from `data/missions.json`
- in-game HUD with mission objective, reward, distance and player state
- procedural street detail: lane markings, crosswalks, lit windows, doors, rooftop pieces and clearer mission beacons
- more readable low-poly player and vehicle silhouettes

Note: the city blockout is generated by `WorldBuilder.gd` at runtime. The editor viewport may look sparse until you run `Main.tscn`.

Controls:

- `WASD` move / drive
- `Shift` run
- `F` enter or exit vehicle
